Business Model des Logiciels Libres

Acteurs principaux

  • Fondations (Apache, Eclipse) : pilotent le développement
  • Distributeurs (Redhat, Mandriva) : packaging et support
  • Éditeurs : créent des produits open source
  • Prestataires : services d'intégration et support

Modèles économiques

  • Support payant : abonnements pour assistance technique
  • Double licence : version libre + version pro avec fonctionnalités supplémentaires
  • Services associés : formation, hébergement, personnalisation
  • Financement communautaire : dons, sponsoring

Exemples concrets

  • Redhat : 82% de revenus via les abonnements support
  • MySQL : modèle de double licence (GPL + licence commerciale)
  • GitHub : plateforme gratuite + options payantes pour entreprises
